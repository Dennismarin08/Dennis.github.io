Sistema de Gestión de Parqueaderos 🚗

Un sistema web completo para la administración de parqueaderos. Diseñado para hacer más fácil el control de vehículos, cobros y clientes frecuentes, con una interfaz intuitiva y moderna.

✨ Funcionalidades principales

  • 🔑 Login con roles (admin y trabajador).
  • 🅿️ Registro de entradas y salidas de vehículos.
  • 💰 Control de caja con apertura y cierre diario.
  • 📊 Reportes gráficos de ingresos y flujo de vehículos.
  • 📂 Historial filtrado por fechas y tipo de vehículo.
  • 👥 Membresías mensuales para clientes frecuentes.
  • 📤 Exportación de reportes a Excel/PDF.

⚙️ Tecnologías usadas

Frontend: HTML, CSS, JavaScript
Backend: PHP
Base de datos: MySQL
